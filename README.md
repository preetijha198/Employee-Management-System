Employee Management System

Overview

The Employee Management System is a web application that allows users to manage employee records efficiently. It provides functionalities to add, edit, and delete employee records with details such as name, email, salary, and joining date.

Features

Add Employees: Add new employee details.

Edit Employees: Modify existing employee records.

Delete Employees: Remove employee records.

View Employee List: Displays all employees in a structured table format.

Technologies Used

Frontend: React.js, Axios, CSS template, sweetalert2
Install dependencies

npm install
npm start

Open http://localhost:3000 in your browser.
